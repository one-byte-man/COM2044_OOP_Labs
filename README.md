# Ankara University - COM2044 Object-Oriented Programming Labs

This repository contains the laboratory assignments and sample solutions for the **COM2044 Object-Oriented Programming** course at **Ankara University, Department of Computer Engineering**.

## Tech Stack & Environment

- **Language:** Java
- **JDK Version:** JavaSE-21
- **IDE:** Eclipse IDE (2025-09)
- **Author:** Ahmet Akpinar ([one-byte-man](https://github.com/one-byte-man))

## Educational Use Only

These solutions are shared for **learning and reference purposes only**.

- Please **do not copy-paste** these solutions for your own assignments.
- Academic integrity is essential. Try solving the problems yourself first to understand core concepts.

---

## Laboratory Works

### Lab 1 - Bank Account Management System

- **Key Concepts:** Classes, constructors, instance variables, `Scanner` class
- **Description:** A menu-driven simulation of basic banking operations such as deposit, withdraw, and balance inquiry.
- **Files:** `Account.java`, `AccountTest.java`

### Lab 2 - Mixed Diamond Pattern

- **Key Concepts:** Nested loops, decision structures, logical operators, modulus
- **Description:** Generates a symmetric diamond pattern that alternates between palindromic numbers (odd rows) and asterisks (even rows).
- **Files:** `DaimondPattern.java`

### Future Labs

This repository will be updated with upcoming lab assignments throughout the semester.

## How to Run

1. **Clone the repository**

	```bash
	git clone https://github.com/one-byte-man/COM2044_OOP_Labs.git
	```

2. **Import into Eclipse IDE**

	Open Eclipse IDE and import the project folder.

3. **Configure Java version**

	Ensure the project build path is set to **JavaSE-21**.

4. **Run the programs**

	Run the `main` methods in `AccountTest.java` or `DaimondPattern.java`.
